# Telegram Actions

Provides some actions for ease of use with Telegram.

## Current Features

- Mass Delete your messages from multiple Chats

## Installation

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

## Releasing

Create Tag:

```bash
git tag -a vX.X.X -m "Release vX.X.X"
```

Push Tag:

```bash
git push origin vX.X.X
```
